# Heart Disease Prediction System

A full-stack machine learning application for predicting heart disease risk.

## Tech Stack

- FastAPI
- HTML
- CSS
- JavaScript
- Scikit-Learn
- KNN

## Features

- Heart disease risk prediction
- FastAPI backend
- Responsive frontend
- Real-time inference

## Run Backend

uvicorn api:app --reload

## Model

KNN Classifier trained on heart disease dataset.
